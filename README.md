# Hindu-game

## 1. Objetivo do Jogo
O jogador deve desenvolver uma pequena civilização hindu ao longo de 5 minutos. O objetivo é
coletar recursos, evoluir ferramentas, construir estruturas e obter bênçãos religiosas para alcançar
o maior nível possível de civilização antes que o tempo termine.

## 2. Duração da Partida
A partida dura aproximadamente 5 minutos, ideal para uma feira escolar. O jogo é rápido e focado
em decisões estratégicas simples.

## 3. Recursos do Jogo
**Os principais recursos coletados são:**
- Arroz: usado para alimentar a população e criar trabalhadores.
- Madeira: usada para construções básicas.
- Pedra: utilizada para templos e estruturas avançadas.
- Especiarias: recurso raro usado para melhorias especiais.
 
## 4. Trabalhadores
O jogador começa com 3 trabalhadores. Eles podem ser enviados para coletar recursos no mapa.
Cada coleta leva entre 5 e 8 segundos. Mais trabalhadores podem ser treinados usando comida.

## 5. Evolução de Ferramentas
**As ferramentas aumentam a eficiência da coleta de recursos:**
- Ferramentas de Pedra (inicial)
- Ferramentas de Bronze (+30% produção)
- Ferramentas de Ferro (+60% produção)
Essa mecânica representa o avanço tecnológico da civilização.

## 6. Construções
**O jogador pode construir diferentes estruturas:**
- Fazenda: produz mais comida.
- Pedreira: aumenta produção de pedra.
- Oficina: permite melhorar ferramentas.
- Templo: desbloqueia mecânicas religiosas.
  
## 7. Sistema de Religião
**A religião é uma mecânica importante. Construindo templos, o jogador pode receber bênçãos de
deuses inspirados na cultura hindu. Exemplos:**
- Indra: dobra produção de comida por um tempo.
- Agni: aumenta produção geral.
- Lakshmi: concede recursos extras.
  
## 8. Unidade Especial
O jogador pode desbloquear um elefante, uma unidade poderosa que pode coletar recursos muito
rapidamente ou construir estruturas quase instantaneamente.

## 9. Eventos Aleatórios
**A cada certo tempo eventos podem ocorrer:**
- Monções aumentam produção de comida.
- Animais selvagens podem atrasar trabalhadores.
- Festivais religiosos dão bônus temporários.
  
## 10. Sistema de Eras
**A civilização evolui durante o jogo:**
- Aldeia do Rio
- Sociedade Védica
- Cidade Sagrada
Cada era desbloqueia novas possibilidades.

## 11. Pontuação Final
**A pontuação final depende de:**
- Nível da civilização
- Número de templos construídos
- Tecnologias desbloqueadas
- Recursos restantes.
O resultado final mostra o nível alcançado pelo jogador.
